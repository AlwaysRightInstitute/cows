## Swift

Swift modules cannot contain resources, hence the cows are compiled into code
cows. Code cows are pretty slow to compile due to the excellence of the
compiler.

## Usage

```swift
import cows

cows.allCows

let someCow = vaca()
```


## API

### allCows()

Returns an array of cows.

### vaca()

Returns a random cow.
